# Афиша KZ — страница "О проекте"

## О проекте
Проект "Афиша KZ" — это веб-платформа для поиска событий в Казахстане.  
Страница `/about` показывает информацию о проекте, как он работает, преимущества платформы и команду.

## Технический стек
- Next.js 14+ (App Router)
- Tailwind CSS
- React / TypeScript
- Lucide Icons для иконок
- Анимации через компонент `fade-in.tsx`

## Структура проекта

app/
├─ about/
│ ├─ page.tsx # главная страница "О проекте"
│ └─ fade-in.tsx # компонент анимации блоков


## Содержание страницы "О проекте"
1. Hero-блок с заголовком и кратким описанием
2. Блок "Как это работает" — 3 шага с иконками
3. Блок "Почему Афиша KZ" — 3 преимущества
4. Блок "Команда" — placeholder-аватары
5. CTA-блок — кнопка "Смотреть афишу"

## Использование Claude Code
Claude Code использовался для:
